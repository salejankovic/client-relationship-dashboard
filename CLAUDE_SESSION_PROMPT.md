## Project Context

**Zlatko** is a production CRM platform built with Next.js 15, React 19, Supabase (PostgreSQL), and integrated with Gmail API and Google Gemini AI. It manages existing clients and prospects for Appworks d.o.o., with features including communication tracking, email import with AI summaries, intelligence feed, and dynamic settings management.

**Key Technologies:** Next.js App Router, TypeScript, Supabase real-time, Gmail OAuth 2.0, Gemini Pro AI, shadcn/ui components, Tailwind CSS.

**Architecture:** Supabase backend with real-time subscriptions, Gmail OAuth for email sync, Gemini for AI summaries, modular component structure with custom React hooks for data management.
